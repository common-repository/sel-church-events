This plugin created for official church themes from Selthemes.com

== Description ==
This plugin registers a custom post type for event items, taxonomy, widgets and metabox.

*This plugin doesn’t show event items in your theme itself.*
*This plugin required 'CMB2' plugin*

== Frequently Asked Questions ==

= Installation Details =
1. Upload 'sel-church-events' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Install required plugin, called 'CMB2'.
1. Add your church events.

=Can I use this plugin with “non-selthemes” themes?=
No, this plugin doesn't show content itself.
